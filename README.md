# Abridge

Unified manager for multiple Coding Agents' configurations and session records.

## Features
- Unified MCP server configuration
- Cross-tool session retrieval
- Multi-device configuration synchronization

## Getting Started
```bash
bun install
bun run dev init
```

## Project Structure
- `packages/core`: Core logic and storage
- `packages/cli`: Command-line interface
- `packages/sdk`: Integration SDK
